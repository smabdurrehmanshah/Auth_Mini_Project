# 🔐 Auth Mini – MERN Authentication Project

A full-stack authentication system built with the **MERN Stack** that includes secure user authentication, profile image upload, and API caching for improved performance.

This project demonstrates how modern web applications handle authentication, file uploads, and performance optimization.

---

## 🚀 Tech Stack

**Frontend**
- React (Vite)
- Axios
- Context API / State Management

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose

**Additional Services**
- Cloudinary (for profile image upload)
- Redis (for API caching and performance optimization)

---

## ⚡ Features

- User registration
- User login
- Password hashing with bcrypt
- JWT authentication
- Profile image upload using Cloudinary
- Redis caching for optimized API responses
- Secure REST APIs
- Environment variable configuration
- Separation of frontend and backend
